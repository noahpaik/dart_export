# v0.1.0 (2026-02-24)

첫 공식 릴리스입니다. Step8 파이프라인 안정화와 Track C strict 게이트 구축을 중심으로 정리했습니다.

## Highlights

- Step8 Track C 운영정책 고도화
  - 기본: Track C 미존재 시 `info`로 스킵
  - 엄격: `--step8-strict-trackc`에서 미파싱 즉시 실패
  - 상태 출력: `trackc_mode`, `track_c.source`
- Track C 입력 보강
  - `document.xml`에 `*_pre.xml`이 없으면 `fnlttXbrl.xml` 자동 시도
- Track B fallback 품질 보강
  - XML 문서 분류/파싱 강화
  - 부문매출 테이블 품질 필터, 단일사업부문 정상 스킵 처리
- Step8 요약 JSON 고정
  - 기본 산출물: `<output>_summary.json`
  - 스키마 버전: `schema_version=1.0`
- 테스트/CI 체계 확장
  - 단위 테스트 세트 추가 (`17 tests`)
  - Track C 고정 샘플 strict 게이트 추가
  - 온라인 strict 게이트 스크립트/CI 옵션 추가
